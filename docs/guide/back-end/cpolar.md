
## 安装
`brew tap probezy/core && brew install cpolar`

## 认证 

``` javascript  
 //登录cpolar官网 查看自己的认证token，之后将token贴在命令行里
cpolar authtoken xxxxxxx
 
 ```
## 安装服务
    `sudo cpolar service install`


## 启动服务 // "start" "stop" "restart" "install" "uninstall"
`sudo cpolar service start`
