## Ningx
### mac安装  
brew install nginx

启动 nginx || sudo nginx  
重启 sudo nginx -s reload  
停止 nginx -s stop quit || killall nginx  
查看nginx版本  
nginx -v  
查找nginx配置文件所在位置 
nginx -t  
查找nginx安装位置  
brew info nginx  
![alt](/img_1.png)  
检查是否启动成功  
ps -ef |grep nginx
